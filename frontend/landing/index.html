From nothing to a first result in three commands.

No account, no signup, no lock-in. It installs in seconds, runs on your own machine, and writes everything it does to disk where you can read it.

1

Install it

$ curl -fsSL https://openscience.sh/install | bash

or npm install -g @synsci/cli

2

Point it at your work

$ cd my-project && synsci

Bring your own model. Aim it at a repo, a dataset, or an empty folder.

3

Ask a real question

synsci ❯ is E545K a kinase-domain mutation?

It plans, reads the sources, runs the code, and records a result you can reproduce.

macOS · Linux · Windows AGPL-3.0 · runs locally · no telemetry by default · read the source
